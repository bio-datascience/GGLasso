Welcome to GGLasso's documentation!
===================================

``GGLasso`` is a Python package that enables sparse inverse covariance estimation (Graphical Lasso). Moreover, it contains solvers and model selection procedures for Multiple Graphical Lasso problems such as Group and Fused Graphical Lasso.

The package is available on `Github`_ .

.. _Github: https://github.com/fabian-sp/GGLasso
